# 课程表小程序 - 开发计划

## 一、核心功能

### 1. 课程管理
- [x] 手动添加课程（课程名称、授课教师、上课地点）
- [x] 设置开始上课时间（课表开始的第一天）
- [x] 设置上课时间（星期几、第几节、每次上课连续几节）
- [x] 设置课程时长（每节课上课时间、下课时间、课间休息时长）
- [x] 设置每天课程节数（每天上午、下午、晚上上几节课）
- [x] 设置每学期教学周数（如 1-18 周）
- [x] 手动调整每门课程的上课周数（如仅在单周/双周/指定周上课）
- [x] 设置是否显示周末
- [x] 编辑课程信息
- [x] 删除课程
- [x] 课程颜色自定义（为每门课程分配不同颜色以便区分）


### 2. 课程表导入
- [x] 手动输入编辑
- [x] 通过图片（OCR 识别）导入完整课程表
- [x] 通过文件（Excel/CSV/JSON）导入完整课程表
- [x] 导入后预览并确认/手动修正识别结果

### 3. 上课提醒通知
- [ ] 上课前通过订阅消息/通知提醒用户
- [ ] 自定义提醒时间（如课前 15 分钟、30 分钟）
- [ ] 开启/关闭通知开关

---

## 二、视图与展示

### 4. 课程表展示
- [x] 周视图：按星期一到星期日展示课程网格
- [x] 日视图：展示当天课程详情
- [x] 支持左右滑动切换周次
- [x] 点击课程卡片查看详情（教师、地点、周数等）
- [x] 当前周次高亮显示
- [x] 非本周课程置灰或隐藏

### 5. 今日课程
- [ ] 首页展示今日课程列表
- [ ] 显示下一节课的倒计时
- [ ] 无课时显示友好提示

---

## 三、学期与周次管理

### 6. 学期管理
- [x] 设置当前学期（学期名称、开学日期、教学周数）
- [x] 支持多学期切换与历史学期查看
- [x] 寒暑假/节假日周次标记（这些周不显示课程）
- [x] 调休补课周处理（如国庆调休导致周末上课）

---

## 四、数据管理

### 7. 数据存储
- [x] 使用微信本地存储（wx.setStorageSync）持久化课程数据
- [x] 数据导出为文件（备份）
- [x] 数据导入恢复（从备份文件还原）

## 五、附加功能（可选）

- [ ] 添加考试科目（科目、时间、地点、座位号）
- [ ] 考试倒计时提醒

### 9. 成绩管理（可选扩展）

### 10. 课程统计

### 11. 个性化设置
- [x] 课程表背景图片自定义
- [x] 深色模式支持
- [x] 周末是否显示课程（可隐藏周末）

### 12. 其他实用功能
- [ ] 分享课程表给好友（生成分享卡片）
- [ ] 课程笔记/备注（为每门课程添加备注信息）
